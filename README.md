## Get around

The projects in this repository demonstrate what's possible with Storyblok: how to build, deploy, integrate, and migrate your website. Directories are organized by these use cases and follow the `action/topic-tech` name pattern.

This means that you can find the code for the tutorial **Integrate Algolia Search into Storyblok with Astro** in the `integrate/algolia` directory, and the code for the tutorial **Build a GEO-ready website with Storyblok and Astro** in the `build/geo-astro`.

> [!TIP]
> Storyblok offers [mature SDKs](https://github.com/storyblok/monoblok) for popular frontend frameworks. Feel free to adapt the code examples to the framework or programming language of your choosing.

What can you find in each directory:

| Directory | Description                                                                                                                                              |
| --------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Build**     | Tutorials focused on building with Storyblok. These examples aren't tied to any specific integration and cover best practices and fundamental web development.                                                                                                             |
| **Deploy**    | Tutorials that teach you how to deploy your Storyblok project to different hosting services. This is where you find platform-specific examples.                                                             |
| **Integrate** | Tutorials that help you integrate Storyblok with third-party tools. Some examples cover Storyblok's official apps and plugins, while others focus on standalone services. |
| **Migrate**   | Tutorials that guide you through a migration process from other content management systems to Storyblok. Some examples also include custom tooling to export data from the target CMS.                                                                      |

## Get started

To clone individual projects, open the tutorial directory and follow the instructions in the `README' file, which includes setup information, links, and relevant details.

Explore these tutorials to learn how to take advantage of Storyblok's native features, use it as inspiration for your own project, or compare different implementation approaches.
